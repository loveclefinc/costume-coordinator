# 🎭 Costume Coordinator

スマートな衣装コーディネーション管理アプリ - 音楽イベント参加者のための最適な衣装提案システム

## 📋 概要

Costume Coordinatorは、音楽イベント（オーケストラ、吹奏楽、アンサンブルなど）の参加者に対して、最適な衣装コーディネーションを提案するアプリです。スーツ、ネクタイ、蝶ネクタイ、ワイシャツを統合管理し、AI画像認識と推奨アルゴリズムを使用して、各参加者に最適な衣装を自動提案します。

## ✨ 主な機能

### 衣装管理
- **複合アイテム管理** - スーツ、ネクタイ、蝶ネクタイ、ワイシャツを統合管理
- **AI画像認識** - 衣装写真から色・素材・特徴を自動抽出
- **属性管理** - 色、素材、柄、季節、フィット感などを詳細に記録

### 最適化・推奨
- **スマート推奨** - 参加者の楽器に基づいた最適な衣装提案
- **相性スコア計算** - スーツ、ネクタイ、ワイシャツの相性を自動計算
- **使用履歴学習** - 過去の使用履歴から参加者の好みを学習

### イベント管理
- **参加者管理** - イベント参加者の登録・管理
- **複数デバイス同期** - Supabaseによるリアルタイム同期
- **イベント履歴** - 過去のイベント情報と使用衣装の記録

### データ共有
- **PDF出力** - イベント参加者の衣装情報をPDF形式で出力
- **データエクスポート** - 全データをJSON形式でエクスポート
- **データインポート** - 他のデバイスからデータをインポート

### 多言語対応
- 日本語、英語、中国語（簡体字・繁体字）、韓国語、スペイン語、フランス語、ドイツ語

## 🚀 クイックスタート

### インストール

1. [ダウンロードページ](https://github.com/loveclefinc/costume-coordinator/releases)からAPKをダウンロード
2. ファイルをタップしてインストール
3. アプリを起動

### 基本的な使用方法

1. **衣装を追加**
   - 「衣装を追加」ボタンをタップ
   - アイテムタイプ（スーツ、ネクタイなど）を選択
   - 衣装の詳細情報を入力

2. **イベントを作成**
   - 「イベント」タブをタップ
   - 「新規イベント」をタップ
   - イベント情報を入力

3. **参加者を追加**
   - イベント詳細画面で「参加者を追加」をタップ
   - 参加者情報を入力

4. **最適な衣装を提案**
   - 「最適化」ボタンをタップ
   - 各参加者に対して最適な衣装が提案される

5. **結果を確認・共有**
   - 推奨結果を確認
   - 「PDF出力」ボタンでPDFを生成・共有

## 📱 システム要件

- **OS**: Android 8.0以上
- **RAM**: 2GB以上（推奨 4GB以上）
- **ストレージ**: 100MB以上
- **インターネット**: クラウド同期時に必要

## 🔧 高度な機能

### クラウド同期（Supabase）

複数デバイス間でのリアルタイム同期を実現：

1. Supabaseプロジェクトを作成
2. アプリの設定画面でSupabaseの認証情報を入力
3. 複数デバイスで自動同期

### オフライン同期

インターネット接続がない環境でも動作：

- ローカルストレージにデータを保存
- AirDropでデータを共有
- 接続復帰時に自動同期

### データのエクスポート・インポート

- 「設定」→「データ管理」→「エクスポート」でデータをバックアップ
- 「インポート」で他のデバイスからデータを復元

## 🎵 使用例

### シナリオ1：オーケストラのコンサート

1. コンサートのイベントを作成
2. 全参加者（バイオリン、チェロ、フルートなど）を登録
3. 各参加者の衣装を最適化
4. 推奨されたコーディネートを確認
5. PDFで参加者全員に共有

### シナリオ2：吹奏楽部の定期演奏会

1. 演奏会のイベントを作成
2. 部員全員を参加者として登録
3. 過去の使用履歴から最適な衣装を提案
4. 新しい衣装を試す場合は手動で変更
5. 使用衣装を履歴に記録

### シナリオ3：複数の会場でのツアー

1. 各会場のイベントを作成
2. Supabaseで複数デバイス間で同期
3. どのデバイスからでも最新の情報にアクセス
4. オフラインでもローカルデータで動作
